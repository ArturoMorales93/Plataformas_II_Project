Types of data typically found in corporations

Unstructured Data
- These are found in the body of emails, printed documents, product specifications, PDF files.
Transactional Data
- These are all those data originating from events and functions of the company related to monetary transactions such as sales and purchases, but also to non-monetary transactions, but quantitative ones.
Metadata
- This is data that identifies and or defines other data. These are commonly found in repositories that have some level of formality, such as documents in XML and JSON format, spreadsheets, etc.
Hierarchical Data
- They are data that is stored and related to other data to form information. They can be distributed among different computer data, but these must in some way relate the data they have to form the information.
Reference Data
- This is a type of Master Data, used to categorize data or also to relate data to some important information in the company.
Master Data
- The type of data that has been discussed in this document, this corresponds to the core of the most important data of a company, this in turn has different access interfaces to only the data necessary for the other systems.
