¿Por qué el Internet de las cosas?

Más allá del sensacionalismo y todo lo que se ha criticado con respecto al internet de las cosas, la humanidad está utilizando la tecnología como parte de su diario, con solo mencionar los smartphones que también son dispositivos IOT, el cual estos ya poseen la capacidad de interactuar de manera automática con otros dispositivos, como por ejemplo controlar sensores del hogar, monitorear el estado de un vehículo y con asistentes de voz que utilizan la inteligencia artificial para reconocer nuestras palabras.

Ahora que se tiene como ejemplo el uso de los smartphones, también están otros dispositivos similares como son las pulseras que son muy populares para monitorear algunas partes del cuerpo humano, también las pulseras se están utilizando ahora como medio de pago gracias a que se pueden conectar con una entidad bancaria.

En la industria de los automóviles se ha utilizado para ofrecer funciones avanzadas como el mencionado "piloto automático" de muchas compañías como TESLA. También mencionar que el IOT complementada con otra área como el Machine Learning dan aún más soluciones casi que imposibles o muy difíciles de conseguir hace menos de 5 años.

¿Cómo funciona el Internet de las Cosas?

En resumen, el IOT no tiene un concepto como tal, pero sigue la filosofía de conectar cualquier cosa a internet, desde ropa, accesorios, electrodomésticos, muebles, automóviles, etc. La necesidad de hacer esto, como se mostró en el video, es lograr hacer aquellas tareas que pudieran ser imposibles de realizar.

Básicamente estos dispositivos contienen un pequeño circuito que contiene muy pocos recursos específicos para realizar sus funciones. Entre esos recursos posee un microcontrolador que es el encargado de realizar las operaciones principales como recolectar y enviar datos que reúne de sensores, estos sensores pueden ser de temperatura, humedad, movimiento, etc.

¿Qué necesito para crear mi dispositivo del IOT?

Con los años se han ido fundando comunidades importantes donde se comparten proyectos IOT Open Source, así como GitHub donde se comparte código fuente de muchos sistemas de Software, existen también donde se comparten especificaciones de sensores, microcontroladores y todo aquel conocimiento de electrónica necesaria para poder desarrollar o implementar alguna solución IOT, entre estas se encuentran las siguientes: Hackster, Circuits IO, Upverter, OSHPark, etc.

Si deseamos empezar a desarrollar nuestros propios prototipos utilizando sensores, podemos visitar el sitio web oficial de la plataforma Arduino: Sensors and components Arduino.cc
